# 小红书爆文分析 Agent

> 帮助品牌方运营人员快速分析小红书爆文趋势，拆解爆款要素并生成选题建议

## Project Identity

- **项目名称**: xiaohongshu-trend-agent
- **一句话描述**: 帮助品牌方运营人员快速分析小红书爆文趋势，拆解爆款要素并生成选题建议
- **目标用户（ICP）**: 品牌方运营人员和内容策划，每周需要产出 3-5 篇小红书内容

### What we don't do

- 不做自动发布——只提供分析和建议，不代替用户发布内容
- 不做账号管理——不涉及小红书账号的登录、运营或数据管理
- 不做付费投放优化——不涉及广告投放策略或 ROI 分析


## Single Core Interaction

用户输入品类关键词（如'美妆'）或竞品账号 → Agent 检索近期爆文 → 拆解标题/封面/钩子/标签/CTA 五要素 → 输出趋势报告 + 3 条选题建议

## Tech Stack

| 层 | 选型 |
|---|---|
| 语言 | Python |
| 框架 | AgentScope |
| LLM | DashScope Qwen |
| 数据源 | 小红书搜索 API / 第三方爬虫 |

## Data Flow

1. 用户输入品类关键词或竞品账号
2. Agent 调用搜索 API 检索近 30 天爆文（点赞 > 1000）
3. 对每篇爆文提取标题、封面、正文钩子、标签、CTA
4. LLM 分析共性模式，拆解为五要素趋势报告
5. 生成 3 条可执行的选题建议，输出结构化报告


## Compliance

- 仅采集小红书公开可见内容，不爬取私密或仅粉丝可见内容
- 不存储笔记作者的个人 ID、手机号等隐私信息
