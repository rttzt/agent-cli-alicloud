# AGENTS.md — xiaohongshu-trend-agent

## Project Identity

小红书爆文分析 Agent：帮助个人博主通过品类关键词分析小红书热门内容规律，输出结构化的爆文要素拆解报告，辅助选题和内容策划。

## Single Core Interaction

用户输入品类关键词（如"美妆"）→ Agent 检索小红书公开热门笔记 → 分析爆文共性要素（标题、封面、结构、互动） → 输出结构化拆解报告。

## Tech Stack

| 层 | 选型 |
|---|---|
| 框架 | AgentScope |
| LLM | 通义千问（DashScope） |
| 数据源 | 小红书公开搜索 API / 第三方数据服务（用户配置） |
| 部署 | 函数计算 / AgentRun（Roadmap） |

## Data Flow

    用户输入关键词
        → 调用数据工具获取公开热门笔记列表
        → LLM 分析笔记共性（标题模式/封面规律/内容结构/互动特征）
        → 结构化输出爆文要素拆解报告

## Compliance

- 仅分析小红书公开可见内容
- 不爬取私域数据（私信、未公开笔记、粉丝列表）
- 不存储任何个人 ID 或用户画像
- 不生成虚假数据或伪造互动数据
- 不抄袭/复制原文内容
